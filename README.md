# SWorld阅读

#### 介绍
SWorld阅读采用C#开发，结合爬虫技术、多线程技术爬取小说。软件具有小说查找、书架、下载、设置等功能。

#### 软件架构
软件采用如下框架。

MySQL数据库框架：Chloe 3.9.0（该部分在项目中未使用，用于大量小说信息爬取入库操作）

C#爬虫框架：HtmlAgilityPack 1.11.12


#### 安装教程

1. 软件不需要安装，在程序源码的Debug或者Release目录中可直接打开运行，也可以在码云的Release界面下载最新发布版本
